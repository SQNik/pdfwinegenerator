🔧 Instrukcje Testowania Zakładki "Pola":

  1. Kliknij link "⚙️ Pola" powyżej
  2. Sprawdź interfejs: Upewnij się, że tabela pól i przyciski są widoczne
  3. Dodaj pole: Kliknij "Dodaj Pole" i wypełnij formularz
  4. Sprawdź podgląd: Sprawdź czy podgląd formularza/tabeli się aktualizuje
  5. Testuj edycję: Kliknij "Edytuj" przy istniejącym polu
  6. Testuj eksport: Kliknij "Eksportuj Konfigurację"

⌨️ Skróty Klawiszowe:

  • Ctrl + 1: Przejdź do sekcji "Wina"
  • Ctrl + 2: Przejdź do sekcji "Listy Win"
  • Ctrl + 3: Przejdź do sekcji "Import"
  • Ctrl + 4: Przejdź do sekcji "Pola" (nowy!)

🐛 Rozwiązywanie Problemów:

  • Jeśli link nie działa: Sprawdź konsolę przeglądarki (F12)
  • Jeśli sekcja nie ładuje się: Upewnij się że serwer działa
  • Błędy JavaScript: Sprawdź czy wszystkie pliki są poprawnie załadowane
