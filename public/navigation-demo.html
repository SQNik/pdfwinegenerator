NavigationComponent - Demo & Przykłady

Prezentacja różnych konfiguracji wspólnego komponentu nawigacji

Domyślna konfiguracja (Primary Theme)

Podstawowa konfiguracja używana na stronie głównej z dropdown menu.

const navigation = new NavigationComponent({ backgroundColor: 'navbar-wine-primary', textColor: 'navbar-dark', brandText: 'Wine Management System', brandIcon: 'bi-grape', logoUrl: null, showDropdown: true }); navigation.render('#nav-demo-1');

Secondary Theme z Custom Buttons

Konfiguracja używana na stronie win z niestandardowymi przyciskami zamiast dropdown.

const navigation = new NavigationComponent({ backgroundColor: 'navbar-wine-secondary', textColor: 'navbar-dark', brandText: 'Wine Management System', brandIcon: 'bi-bottle', showDropdown: false, customButtons: [ { action: 'backup', icon: 'bi-download', text: 'Backup', class: 'btn-outline-light' }, { action: 'settings', icon: 'bi-gear-fill', text: 'Ustawienia', class: 'btn-outline-light' } ] }); navigation.render('#nav-demo-2');

Info Theme z Custom Dropdown

Konfiguracja z motywem info i niestandardowym dropdown menu.

const navigation = new NavigationComponent({ backgroundColor: 'navbar-wine-info', textColor: 'navbar-dark', brandText: 'Collections Manager', brandIcon: 'bi-folder', showDropdown: true, dropdownItems: [ { id: 'export', action: 'export', icon: 'bi-download', text: 'Export kolekcji', isDivider: false }, { id: 'divider1', isDivider: true }, { id: 'backup', action: 'backup', icon: 'bi-shield-check', text: 'Backup', isDivider: false } ] }); navigation.render('#nav-demo-3');

Instrukcje integracji

Jak zintegrować komponent w swoim projekcie:

<!-- 1. Dodaj kontener --> <div id="navigation-container"></div> <!-- 2. Dołącz skrypt --> <script src="js/components/NavigationComponent.js"></script> <!-- 3. Inicjalizuj --> <script> document.addEventListener('DOMContentLoaded', function() { const navigation = new NavigationComponent({ backgroundColor: 'navbar-wine-primary', brandText: 'Moja Aplikacja', brandIcon: 'bi-house' }); navigation.render('#navigation-container'); window.navigationComponent = navigation; // Obsługa wydarzeń document.addEventListener('navigationAction', function(e) { console.log('Akcja:', e.detail.action); }); }); </script>
